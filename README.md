# MFE-B (User Dashboard)

Micro-frontend for the User role.

## Features
-   Exposes `UserDashboard` component.
-   Fetches protected data from `/api/user/data`.

## Setup
1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Start the app:
    ```bash
    npm run dev
    ```
    Runs on port `3002`.
